Einleitung:
Die Zoo-App informiert Besucher und Zoomitarbeiter/-direktoren über zwei Sonderzuchtprogramme von stark gefährdeten Tierarten. Über die App bekommen die Besucher nähere Informationen über die gefährdeten Tiere, die Zuchtprogramme und werden so für den Arten/-Naturschutz sensibilisiert. Für die Zoomitarbeiter/-direktoren bietet die App eine Übersicht über die Einnahmen/Ausgaben des Zoos, die Besucherzahlen, die Zuchtergebnisse und mögliche Auswilderungen.
Ist es möglich mit stark gefährdeten Tieren, die es in anderen Zoos nicht gibt, die Einnahmen eines Zoos zu steigern? Sind Sonderzuchtprogrammd von bedrohten Tierarten in Zoos erfolgreich und helfen Sie, den Rückgang der Population aufzuhalten? Diese Fragen sollen Mithilfe dieser App beantwortet werden. 
    
Technische Grundlagen:
Kontext der Arbeit: - Es soll eine Zoo App eigenhändig durch Codierung erstellt werden
                    - Es soll eie solide, robuste, fehlerfreie und gut dokumentierte Software entwickelt werden

Technische Grundlagen:

-   der Quellcode wird mithilfe von Visual Studio Code ertellt und mittelts Git verwaltet
-   es wird ein Projekt-Exposé in Canva erstellt
-   es wird ein Frontend und ein Backend benötigt
-   es werden viele verschiedene Funktionen in den Code eingebunden und Frameworks verwendet

Beschreibung der Softwarearchitektur:
-   der Quellcode wird in Visual Studio Code erstellt/programmiert
-   es wird ein Repository in Github erstellt, wo regelmässig alle Änderungen als commit hochgeladen werden
-   es wird ein Bulma CSS-Framework verwendet
-   die App wird mithilfe von html, css, javascript, img-Dateien, package.json, readme.Datei erstellt
-   es werden Diagramme programmiert und eingebunden

Zusammenfassung und Ausblick:
-   das Frontend könnte noch mit CSS-Styling-Elementen verbessert werden
-   es könnte noch eine Datenbank als Grundlage für die 3 Diagramme erstellt werden
